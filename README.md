#### Windows

1. **Open Command Prompt (cmd)**

2. **Navigate to the project directory**

   ```cmd
   cd path\to\your-repository
   ```

3. **Run the script**

   ```cmd
   python your_script_name.py
   ```

1. **Open Terminal**

2. **Navigate to the project directory**

   ```bash
   cd /path/to/your-repository
   ```

3. **Run the script**

   ```bash
   python3 your_script_name.py
   ```
